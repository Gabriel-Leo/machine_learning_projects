# Drug Consumption Risk Prediction and Feature Importance Analysis

This project explores the use of machine learning techniques to predict groups at risk of Meth consumption based on demographic and psychological assessment data. In addition to predictive modeling, the project also analyzes feature importance in order to better understand which variables contribute most significantly to the prediction process.

The primary goal of the project is not only to evaluate model performance, but also to examine how demographic characteristics and personality-related traits may influence substance use risk patterns.

## Project Objectives
- Train and evaluate multiple machine learning classification models
- Predict Meth consumption risk groups using demographic and psychological features
- Address class imbalance using resampling techniques
- Compare model performance using cross-validation and F1-macro scoring
- Analyze feature importance across models
- Explore relationships between psychological traits and substance use risk

## Models Used
- KNN
- Decision Tree
- Random Forest
