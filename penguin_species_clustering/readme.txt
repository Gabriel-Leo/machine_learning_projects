# Penguin Species Clustering

This project applies unsupervised machine learning techniques to discover natural groupings in penguin data based on physical characteristics such as culmen length, culmen depth, flipper length, and body mass.

Three different clustering algorithms are implemented and compared to evaluate how effectively they identify potential penguin species groupings within the dataset.

## Features
- Data cleaning and preprocessing
- Outlier detection and removal using the IQR method
- Feature scaling
- Clustering visualization using PCA dimensionality reduction
- Comparison of multiple clustering algorithms

## Models Used
- K-Means
- Mean Shift
- DBSCAN

## Technologies Used
- Python 3.13.5
- Pandas 2.3.2
- Scikit-learn 1.7.1
- SciPy 1.16.0
