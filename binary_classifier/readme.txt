# Binary Classification: Diabetes Prediction

This project trains and evaluates three machine learning models for a binary classification task: predicting whether a patient is diabetic or non-diabetic based on medical variables.

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest

## Tech Stack
- Python: 3.13.5  
- NumPy: 2.3.1  
- Pandas: 2.3.2  
- Scikit-learn: 1.7.1  
- SciPy: 1.16.0  
- Matplotlib: 3.10.5
