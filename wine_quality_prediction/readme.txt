# Regression: Wine Quality Prediction

This project explores the use of regression techniques to predict wine quality based on its physicochemical properties. Using machine learning models trained on wine characteristic measurements, the objective is to estimate wine quality scores as accurately as possible while comparing the strengths and limitations of different regression approaches.

The project also investigates how feature engineering and dimensionality reduction techniques affect model performance and generalization.

## Project Objectives
- Predict wine quality scores using regression models
- Compare the performance of multiple regression techniques
- Evaluate the impact of polynomial features and dimensionality reduction

## Models Used
- Ridge Regression
- Polynomial Regression
- Principal Component Regression (PCR)
- Polynomial + Principal Component Regression

## Machine Learning Techniques
- Feature Correlation Analysis
- Standardization and Scaling
- Polynomial Feature Engineering
- Principal Component Analysis (PCA)
- Cross Validation
- Hyperparameter Tuning
- Regression Evaluation Metrics
- Evaluation Metrics
