# Multiclass Classification: Meth Consumption Risk

This project trains and evaluates three machine learning models for a multiclass classification task: predicting the risk for individuals to fall into meth consumption.

## Models Used
- KNN
- Decision Tree
- Random Forest

## Tech Stack
- Python: 3.13.5  
- Pandas: 2.3.2  
- Scikit-learn: 1.7.1  
- SciPy: 1.16.0  
- Matplotlib: 3.10.5 
- Seaborn: 0.13.2
- imbalanced-learn: 0.14.0
