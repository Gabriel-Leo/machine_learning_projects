# Quality Control: Quality Control Tests Optimization

This project focuses on optimizing a manufacturing quality control process using machine learning techniques. The objective is to develop a predictive model capable of classifying production batches as either good or poor quality based on the results of 30 quality control tests.

By analyzing the predictive importance of each test, the project aims to identify the minimum number of tests and decision rules required to maintain reliable quality predictions. Any tests that contribute little or no predictive value may potentially be removed, reducing inspection time and overall quality control costs.

## Project Objectives
- Predict batch quality using quality control measurements
- Minimize the number of decision rules used by the model
- Identify which quality control tests contribute most to prediction accuracy
- Reduce unnecessary testing and optimize inspection costs
- Evaluate model performance using cross-validation
