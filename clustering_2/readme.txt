# Clustering: Marketing Campaign

The objective of this code is to train a clustering machine learning algorithm to classify client groups according to the data. Three different clustering algorithms will be tried.

## Models Used
- K-Means
- Main Shift
- DBSCAN

## Tech Stack
- Python: 3.13.5
- Pandas: 2.3.2
- NumPy: 2.3.1
- Scikit-learn: 1.7.1
- Scipy: 1.16.0
- Matplotlib: 3.10.5
- Seaborn: 0.13.2
