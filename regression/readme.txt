# Regression: Wine Quality Prediction

The objective os this code is to train a machine learning algorithm in predicting the quality of the wine depending on its properties.

## Models Used
- Ridge Regression
- Polynomial Regression
- Principal Component Regression
- Polynomial + Principal Component Regression

## Tech Stack
- Python: 3.13.5
- Pandas: 2.3.2
- Scikit-learn: 1.7.1
- Matplotlib: 3.10.5
- seaborn: 0.13.2
